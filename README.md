# PastForward: Interactive Historical Simulation

PastForward is a web-based educational simulation that lets users experience the role of Maharana Pratap during the Battle of Haldighati (1576). Users make strategic decisions and see the short-term and long-term consequences of their choices, while comparing them with historical outcomes.

---

## Features

- Interactive decision-making based on historical events  
- Short-term and long-term outcome analysis for each choice  
- Shows alignment or difference with Maharana Pratapâ€™s actual decision  
- Clean, parchment-inspired UI with royal brown accents  
- Fully responsive design for desktop and mobile  
- Built using Python Flask, HTML, and CSS  
- No external images required  

---

## Tech Stack

- **Backend:** Python, Flask  
- **Frontend:** HTML5, CSS3, Jinja2 templates  
- **Design:** Gradient backgrounds, borders, typography, hover effects  
- **Responsive:** Works on all screen sizes
